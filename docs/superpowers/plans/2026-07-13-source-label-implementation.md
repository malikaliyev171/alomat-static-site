# Source Label Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Display concise uppercase source names while preserving every story's actual source URL.

**Architecture:** Add one hostname-to-label helper beside the existing URL sanitizers in `app.js`. `getStorySourceLabel` will use that helper for ordinary and hosted-platform domains, while retaining the current AI Digest and source-text fallback behavior.

**Tech Stack:** Vanilla JavaScript, Node test runner, static-site build script.

## Global Constraints

- Do not change the source button URL.
- Ordinary domains use their registrable domain label.
- WordPress, Substack, Medium, Blogspot, and GitHub Pages use the tenant subdomain.
- Every displayed source label is uppercase.
- Invalid or missing URLs fall back to the supplied source text.

- [ ] **Step 3: Implement the minimal label parser**

Add platform and multipart-suffix sets, then derive the display label:

```js
const HOSTED_PUBLISHING_DOMAINS = new Set(["wordpress.com", "substack.com", "medium.com", "blogspot.com", "github.io"]);
const MULTIPART_PUBLIC_SUFFIXES = new Set(["co.uk", "com.au", "com.tr", "com.uz", "co.jp"]);

function getSiteLabelFromUrl(value) {
  const href = sanitizeStoryUrl(value);
  if (href === "#") return "";
  const labels = new URL(href).hostname.toLowerCase().replace(/^www\./, "").split(".").filter(Boolean);
  if (labels.length === 1) return labels[0].toUpperCase();
  const hostname = labels.join(".");
  const hostedDomain = [...HOSTED_PUBLISHING_DOMAINS].find((domain) => hostname.endsWith(`.${domain}`));
  if (hostedDomain) return labels[labels.length - hostedDomain.split(".").length - 1]?.toUpperCase() || "";
  const suffix = labels.slice(-2).join(".");
  const index = MULTIPART_PUBLIC_SUFFIXES.has(suffix) ? labels.length - 3 : labels.length - 2;
  return (labels[index] || labels[0]).toUpperCase();
}
```

Use `ALOMAT` for AI Digest, use `getSiteLabelFromUrl(story.url)` for ordinary stories, and uppercase the supplied source fallback.
